Jupyter Notebook: Sentiment analysis Apr 2020.ipynb

Required inputs in the same directory:
1) Text files from /risk_factors containing data from 10-Q and 10-K reports
	- file naming convention must follow: TICKER_YYYYMMDD_10Q.txt or TICKER_YYYYMMDD_10K.txt
2) Sentiment dictionary: LoughranMcDonald_MasterDictionary_2018.csv
	- source: Software Repository for Accounting and Finance, University of Notre Dame
	- https://sraf.nd.edu/textual-analysis/code/
	- please update as needed

Outputs:
Charts tracking sentiment changes across multiple periods
One chart per ticker according to input files
